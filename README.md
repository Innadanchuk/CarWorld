# CarWorld

Java Script и jQuery настолько влились в нашу жизнь, что любой пустяковый обработчик событий мы можем отшлифовать, 
вытянув элемент по id или классу. 
Но есть случаи, когда применить Java Script нельзя (по каким-либо желаниям заказчика), 
невозможно (отсутствием установленного скрипта) или просто не нужно. 
Этот пример именно для такого случая: ПРИМЕНЕНИЕ CSS без Java Script.
Предназначен он для того, чтобы продемонстрировать связи родитель-потомок в HTML и СSS.

Итак, в данном примере мы используя контролы Bootstrap 3, создали одностраничный сайт с навбаром, футером, навигацией по странице сайта, 
всплывающим модальным окном и toggle switch в navbar. Toggle Switch предназначен для переключения цвета всего сайта c green to orange.
Весь, вышеперечисленный, функционал был реализован только при помощи HTML и СSS, БЕЗ ИСПОЛЬЗОВАНИЯ java script.
